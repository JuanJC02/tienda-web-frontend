# Tienda de Empanadas - Proyecto Semestral

Este repositorio contiene el desarrollo de una plataforma web diseñada para una tienda de empanadas, realizada como proyecto académico para la asignatura de **Desarrollo Web**.

## 🛠️ Tecnologías utilizadas
* **Frontend:** React.js
* **Lenguajes:** HTML5, CSS3 y JavaScript (ES6+)
* **Estilos:** CSS Modules / Styled Components (opcional según tu uso)

## 📋 Características
* Interfaz responsiva y amigable para el usuario.
* Catálogo interactivo de productos.
* Gestión de carrito de compras.
* Navegación optimizada mediante componentes de React.
